Fake OS

  • There is a start button which can be clicked to reveal a menu displaying the same three applications present on the desktop.
Morse Code Translator
Technologies Used: HTML, CSS/SCSS, JAVASCRIPT & JEST
  • A user can enter either Morse code or English text into the input text area on the left and a translation will appear in the output text area on the right once, the "Encode/Decode" button has been selected.
  • The program can detect whether English or Morse code has been entered by the user and will highlight the input language detected. The output language will also be highlighted.
Google Books
Technologies Used: REACT
  • This webpage allows a user to search for a book inside the google books database.
  • The page will display a max of 10, 20, 30 or 40 results depending on the users input for Max Results.
  • The results will show an image of the book, title, author or authors and a description, if this data is available.
eCommerce Site

Hangman
Technologies Used: JAVA
  • This is a version of the game Hangman that can be played in a Java console application.
  • The game randomly selects a word from a provided list in a text file when the game begins.
  • The user will then be asked to enter a letter or a word. If the letter entered by the user is in the selected word the letter's location in the word will be shown.
  • If the letter is not in the word or the word guess is incorrectly, a part of the hangman will be printed to the console.
Minesweeper
Technologies Used: JAVA
  • This is a version of the game Minesweeper that can be played in a Java console.
  • A 10x10 grid is displayed to the user. They have the option to select, set a flag on, or unset a flag at a grid position
  • The game will end if the user selected a mine.
  • If the user selected a blank coordinate the number of adjacent minds or dashed will display in the grid
